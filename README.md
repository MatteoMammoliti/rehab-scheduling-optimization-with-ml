# 🏥 Rehab Scheduling Optimization with Machine Learning

This Thesis Project tackles the **Rehabilitation Scheduling Problem (RSP)** by integrating **Machine Learning** predictions with the **Answer Set Programming (ASP)** logic-deductive paradigm. The goal of this **AI NeuroSymbolic approach** is to predict the number of feasible assignments to optimize and stabilize the schedules generated by logic solvers.

In clinical environments (such as the Maugeri Institutes), the growing number of logistical and clinical constraints makes manual or purely logic-based scheduling computationally expensive. This "neuro-symbolic" approach leverages predictive models to estimate the real workload capacity for a given day, guiding the ASP solver toward more robust, balanced, and practically applicable solutions.
